Savina Lutfia Nasta’in

Tentang Saya

Hallo, perkenalkan nama saya Savina Lutfia Nasta’in, biasa dipanggil Savina. saya di lahirkan pada tanggal 16 Juli 1998 di Ponorogo. Usia saya sekarang 22 tahun. Saya bertempat tinggal di alamat Jl. Mayjend D.I Panjaitan No. 15 Rt.01 Rw.03 lingkungan krajan purbosuman, ponorogo. Saya Mahasiswa aktif semester 6 di Universitas Muhammadiyah Ponorogo fakultas Teknik, Program Studi Teknik Informatika angkatan 2018.

Untuk saat ini, sembari saya menjadi mahasiswa di universitas muhammadiyah ponorogo saya juga menyibukkan diri saya untuk bekerja sebagai guru les privat anak SD. Menurut saya sebagai mahasiswa bukan hanya materi atau pendidikan dikelas saja yang dibutuhkan tapi juga langsung terjun ke dunia nyata, meskipun program yang saya ambil Prodi teknik informatika dan saya bekerja di dunia Pendidikan SD. Selain itu saya juga mengasah kemampuan untuk berjualan dengan cara online di semua sosial media, mulai dari peralatan rumah tangga, kebutuhan pokok, pakaian, segala aksesoris dll.

Saya juga mempunyai hobi yaitu, nonton dari segala jenis film, drama, pertunjukan, atraksi, semua itu saya sukai dan saya jadikan sebagai hobi. Dan untuk cita-cita saya ingin menjadi guru, karna yang pertama saya suka berinteraksi dengan orang lain, membantu sesama dan menyampaikan ilmu yang saya punya kepada semua orang.
