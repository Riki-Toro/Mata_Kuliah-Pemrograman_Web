Fitur Deskripsi Fitur
Garansi seumur hidup dalam fitur garansi seumur hidup ini ada yang bisa di klaim dan ada yang tidak bisa. Syarat bisa di klaim dapat dilihat pada informasi S&K.
Multi user fitur ini dapat digunakan atau diakses oleh banyak user di saat yang bersamaan.
Update otomatis seperti halnya software-software yang lain, kami menyediakan update otomatis agar jika ada pembaharuan si user langsung bisa menggunakan fitur-fitur dalam pembaharuan, akan tetapi si user juga bisa menonaktifkan fitur ini jika ingin tetap pada versi lama atau update manual.
Cetak laporan terdapat fitur cetak laporan untuk user yang ingin mencetak laporan untuk periode tertentu.
Notifikasi error jika ada kesalahan dalam sistem software maka akan ada notifikasi eror yang bisa dilihat oleh user.
Ubah tema user dapat mengubah tema sesuai yang diinginkan. Dalam fitur ini terdapat banyak tema-tema yang dapat dipilih user, sehingga user tidak perlu mendownload dari luar.
Try icon terdapat banyak icon-icon yang memrepresentasikan sebuah fitur. untuk informasi lebih lengkap dapat dibaca di informasi icon.
